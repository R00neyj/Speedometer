# Gauge 5단위 가이드 추가 계획

## 개요
- 바늘의 위치를 쉽게 파악할 수 있도록 5단위 간격의 눈금(Ticks) 및 숫자 가이드 추가

## 작업 상세
### 1. `src/components/Gauge.vue` 수정
- [ ] 5단위 눈금 데이터를 생성하는 `computed` 속성(`ticks`) 추가
  - `props.max`를 기준으로 0부터 `max`까지 5단위로 생성
  - 각 눈금의 각도 계산 (현재 바늘 회전 로직과 동일한 `-135도 ~ 135도` 범위)
- [ ] 템플릿에 눈금 렌더링 추가
  - SVG 또는 `div`를 사용하여 원형으로 배치
  - 10단위는 더 길거나 굵게 표시하여 시인성 확보
  - 주요 수치(예: 0, 10, 20, 30, 40)는 숫자 라벨 병행

### 2. 스타일링 및 검증
- [ ] 다크 모드 및 라이트 모드 대응 (기존 `text-on-surface-variant` 등 활용)
- [ ] 바늘과 겹치지 않도록 적절한 `z-index` 설정
- [ ] `props.max` 변경 시 눈금이 동적으로 반응하는지 확인
